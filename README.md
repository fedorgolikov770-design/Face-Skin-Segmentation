# Face Skin Segmentation

Программа для создания маски кожных покровов лица на основе ключевых точек (landmarks).

## Установка

### 1. Установить зависимости

```bash
pip install -r requirements.txt
```

### 2. Скачать модель ключевых точек

Скачайте файл `shape_predictor_81_face_landmarks.dat` и положите его в корень проекта.

## Использование

```bash
python main.py <входное_изображение> [выходное_изображение]
```

### Примеры

```bash
# Результат сохранится как photo_skin.jpg
python main.py photo.jpg

# Указать имя выходного файла
python main.py photo.png result.png
```

## Входные данные

Изображение в формате **png** или **jpeg** с лицом человека.

## Выходные данные

Файл такого же размера и формата, как входной файл, содержащий только пиксели кожного покрова лица. Остальные пиксели — чёрные (0, 0, 0).

## Алгоритм

1. **Детекция лица и ключевых точек** — Dlib HOG-детектор + Shape Predictor (81 точка)
2. **Геометрическая маска** — полигон по контуру челюсти (0–16) и линии лба (68–80)
3. **Цветовая фильтрация** — каскад RGB ∩ (HSV ∪ YCbCr) по Kolkur et al. (2017)
4. **Морфологическое рафинирование** — Closing → Opening → Gaussian Blur
5. **Исключение глаз и рта** — адаптивное по EAR/MAR (учитывает открыты ли глаза/рот)

## Зависимости

- Python 3.10+
- OpenCV (`opencv-python`)
- Dlib (`dlib`)
- NumPy (`numpy`)
